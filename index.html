Projets phares

Alimentation mondiale (FAO)

Python • Données publiques

Problème métier : identifier les pays et régions en sous‑nutrition pour cibler les aides alimentaires.

  • Données FAOSTAT 2013–2017, analyses Python/pandas, visualisations.

Ventes librairie LAPAGE

Python • Statistiques

Problème métier : comprendre le CA et anticiper les tendances de vente d’une librairie en ligne fictive.

  • ~680 000 transactions, tests Chi² / ANOVA, visualisations (barres, courbes, heatmaps).
  • Effet Pareto : minorité de clients = majorité du CA.
  • Tendances saisonnières, influence âge/genre, produits top/flop.
